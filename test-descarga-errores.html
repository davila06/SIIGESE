✅ Sistema de Descarga de Errores

Registros fallidos automáticamente disponibles para descarga

🎯 ¿Qué hace el sistema?

✅ Captura automática: Todos los registros que fallan durante el upload de pólizas se almacenan
✅ Datos originales: Se preservan todos los datos originales de cada registro fallido
✅ Descarga Excel: Se genera automáticamente un archivo CSV descargable con los errores
✅ Información detallada: Incluye número de fila, error específico y todos los datos originales

📋 Contenido del archivo de errores

Cuando hay registros fallidos, el sistema genera un archivo CSV con las siguientes columnas:

Fila | Error | POLIZA | NOMBRE | NUMEROCEDULA | PRIMA | MONEDA | FECHA | FRECUENCIA | ASEGURADORA | PLACA | MARCA | MODELO | AÑO | CORREO | NUMEROTELEFONO

🔧 Cómo funciona

Usuario sube un archivo Excel con pólizas
Sistema procesa cada registro y valida los datos
Registros válidos se guardan en la base de datos
Registros con errores se almacenan en una lista especial
Si hay errores, aparece automáticamente el botón "Descargar Archivo de Errores"
Al hacer clic, se genera y descarga un CSV con todos los datos fallidos

💡 Ejemplo práctico

Escenario: Subes un archivo con 100 pólizas, pero 5 tienen errores (fecha inválida, prima vacía, etc.)

Resultado:
  • ✅ 95 pólizas se guardan correctamente en el sistema
  • ❌ 5 registros fallidos se almacenan para descarga
  • 📥 Aparece botón "Descargar Archivo de Errores"
  • 📋 Al descargar obtienes un CSV con las 5 filas problemáticas + detalles del error
  • 🔧 Puedes corregir los errores en el archivo y volver a subirlo

📁 Información técnica

ℹ️ Formato del archivo: CSV con codificación UTF-8
ℹ️ Nombre del archivo: Errores_Polizas_YYYY-MM-DD_HH-mm-ss.csv
ℹ️ Ubicación: Se descarga automáticamente en la carpeta de descargas
ℹ️ Compatibilidad: Se puede abrir con Excel, Google Sheets o cualquier editor CSV
