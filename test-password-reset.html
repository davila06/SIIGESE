🔐 Test - Sistema de Reset de Contraseña

Prueba la funcionalidad completa de forgot/reset password

1. 🔑 Test Login

Prueba el login para verificar si el usuario requiere cambio de contraseña.

POST http://localhost:5000/api/auth/login

2. 📧 Test Forgot Password

Envía un email de reset de contraseña (simula el envío).

POST http://localhost:5000/api/auth/forgot-password

3. 🔄 Test Reset Password

Nota: Necesitas un token válido de reset (se genera en forgot-password).

POST http://localhost:5000/api/auth/reset-password

4. 🔐 Test Change Password

Nota: Requiere estar autenticado (token JWT).

POST http://localhost:5000/api/auth/change-password

📋 Información del Sistema

✅ Funcionalidades Implementadas:

  • Login con flag RequiresPasswordChange - Detecta primer ingreso
  • Forgot Password - Genera token y envía email (simulado)
  • Reset Password - Valida token y actualiza contraseña
  • Change Password - Cambio de contraseña autenticado
  • Base de datos actualizada - Nueva tabla PasswordResetTokens
  • Email templates - HTML profesionales para notificaciones

🔧 Configuración SMTP:

Para envío real de emails: Actualizar credenciales en appsettings.json

"Email": {
  "SmtpServer": "smtp.gmail.com",
  "SmtpPort": 587,
  "Username": "tu-email@gmail.com",
  "Password": "tu-app-password",
  "FromEmail": "noreply@tuapp.com",
  "FromName": "Tu Aplicación"
}
