❌ Problema Identificado:

Desincronización entre Template y Componente

Las columnas definidas en displayedColumns no coincidían con las definidas en el template HTML:

  • Componente tenía: ['avatar', 'name', 'email', 'roles', 'status', 'lastLogin', 'actions']
  • Template tenía: ['userName', 'fullName', 'roles', 'isActive', 'lastLoginAt', 'actions']

✅ Solucionado:

Se corrigió displayedColumns para coincidir con el template:

displayedColumns: string[] = ['userName', 'fullName', 'roles', 'isActive', 'lastLoginAt', 'actions'];

También se agregó ngAfterViewInit para configurar el paginator correctamente.
