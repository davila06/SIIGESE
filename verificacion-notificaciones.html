📊 Flujo de Datos

Cobros Vencidos

1. Query a tabla Cobros WHERE Estado = EstadoCobro.Pendiente AND IsDeleted = false AND FechaVencimiento < Hoy 2. Calcula días de mora: DiasMora = (Hoy - FechaVencimiento).Days 3. Ordena por FechaVencimiento descendente 4. Retorna CobroVencidoDto[]

Pólizas por Vencer

1. Query a tabla Polizas WHERE IsDeleted = false AND FechaVigencia >= Hoy AND FechaVigencia <= Hoy + DíasAntes (30) 2. Calcula días hasta vencimiento: DiasHasta = (FechaVigencia - Hoy).Days 3. Ordena por FechaVigencia ascendente 4. Retorna PolizaVencimientoDto[]

⚠️ Notas Importantes

  • Cobro.CorreoElectronico - Campo NO existe, ClienteEmail queda vacío
  • Poliza.SumaAsegurada - Campo NO existe, MontoAsegurado = 0
  • Envío de emails - Infraestructura lista, implementación comentada
  • Autenticación - Todos los endpoints requieren [Authorize]
