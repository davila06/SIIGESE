🐛 Debug - Carga de Usuarios

Pruebas de Conectividad:

📋 Pasos para Debuggear:

  1. Verificar Autenticación: Asegúrese de estar logueado
  2. Verificar URL del Backend: Confirmar que el proxy esté funcionando
  3. Verificar Endpoint: /api/users debe responder
  4. Verificar Headers: Token de autenticación debe estar presente
  5. Verificar Roles: Usuario debe tener permisos de Admin

🔍 URLs de Prueba:

  • Frontend Local
  • Backend Azure (directo)
  • Frontend con Proxy
